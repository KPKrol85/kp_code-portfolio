O studio

Subtelna estetyka, bez kompromisów.

Studio Noir powstało z myślą o osobach, które oczekują najwyższej jakości, dyskrecji i perfekcyjnego wykończenia. Łączymy rzemiosło z technologią, tworząc styl dopasowany do Ciebie — nie do trendu.

Historia

Atelier założone przez stylistów z doświadczeniem w modzie i branży urody.
