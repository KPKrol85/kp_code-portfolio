Regulamin (wersja demonstracyjna)

Niniejszy regulamin ma charakter przykładowy i nie stanowi porady prawnej.

1. Postanowienia ogólne

Sklep sprzedaje cyfrowe produkty, które są dostarczane po zaksięgowaniu płatności.

2. Licencje

Zakupione produkty są objęte licencją przypisaną do konta klienta.

3. Zwroty

Z uwagi na charakter cyfrowy zwroty realizowane są zgodnie z obowiązującymi przepisami.
