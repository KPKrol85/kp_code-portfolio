Projekt: e-commerce-02 (KP_Code Digital Vault)

Opis projektu:
e-commerce-02 to frontendowa platforma sprzedaży własnych produktów cyfrowych (UI kits, komponenty, dashboardy, assets, knowledge products) rozwijana jako projekt portfolio klasy produkcyjnej oraz fundament realnego digital product business (KP_Code Digital Vault).

Projekt jest budowany jako SPA w czystym Vanilla JS, bez frameworków, z naciskiem na stabilną architekturę, czytelność kodu i długoterminową skalowalność.

Zakres technologiczny:
– HTML, CSS, Vanilla JavaScript
– Frontend-only
– Architektura SPA (routing hash-based)
– Podejście komponentowe
– Brak frameworków i build stepów

Założenia architektoniczne:
– Jedna wersja rozwiązania = jedna obowiązująca wersja
– Decyzje architektoniczne są świadome i trwałe
– Brak „hacków”, skrótów i tymczasowych obejść
– Kod ma być przewidywalny, czytelny i łatwy do utrzymania

Styl i jakość:
– Nowoczesny, profesjonalny UI (SaaS / Apple / Stripe-level)
– Design system oparty o design tokens
– Obsługa light / dark mode
– Dbałość o UX, dostępność i stabilność

Zasady pracy:
– Zawsze zakładamy pracę lokalnie w folderze projektu (VS Code / Codex)
– Nie tworzymy nowych folderów poza istniejącą strukturą bez decyzji
– Nie zmieniamy ustaleń architektonicznych bez wyraźnej zgody
– Pracujemy spokojnie, krok po kroku
– Jedna zmiana → test → commit

Kod i wyjaśnienia:
– JavaScript tłumaczony logicznie, prosto i edukacyjnie
– Myślenie senior dev: stabilność > spryt
– Preferowane czyste funkcje i separacja odpowiedzialności
– Brak nadmiarowych abstrakcji i „magii”

Commity:
– Krótkie, profesjonalne, po angielsku
– Format:
  feat: ...
  fix: ...
  refactor: ...
– Opis commitów w punktach (*)

Codex:
– Jeśli proszę o prompt dla Codexa:
  – ma być precyzyjny
  – z jasno określonym celem i zakresem zmian
  – bez lania wody
– Nie dodawać propozycji commitów, jeśli nie zostały wyraźnie poproszone

Cel projektu:
– Gotowa do sprzedaży i skalowania platforma digital products
– Fundament pod markę KP_Code Digital Vault
– Projekt referencyjny o jakości produkcyjnej
