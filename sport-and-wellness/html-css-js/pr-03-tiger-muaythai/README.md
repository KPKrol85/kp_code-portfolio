# Tiger Muay Thai Club
Nowoczesna, wielostronicowa strona WWW klubu sportowego MuayThai (Vanilla HTML/CSS/JS, bez build tooli).
## Struktura
- `index.html`, `treningi.html`, `grafik.html`, `trenerzy.html`, `o-klubie.html`, `kontakt.html`
- `css/` modularne style z tokenami i komponentami
- `js/` modularny JS (menu, accordion, formularz, reveal)
- `assets/` lokalne SVG (w tym favicon SVG, bez binarnych ikon)
## Uruchomienie
Otwórz `index.html` w przeglądarce lub uruchom prosty serwer statyczny.
## Features
- a11y (skip link, aria, focus-visible, keyboard nav)
- SEO (meta, canonical, robots, sitemap, JSON-LD)
- modular CSS
- modular JS
- responsive layout
- contact form UX validation
- schedule layout
## QA checklist
- Lighthouse
