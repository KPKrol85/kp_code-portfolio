## npm scripts (from `package.json`)

| Script | Command | What it does | When to use |
|---|---|---|---|
| `build:css` | `postcss ./css/style.css -o ./css/style.min.css && node ./scripts/verify-build.mjs css ./css/style.min.css` | Builds minified CSS from the modular CSS entry and validates output integrity. | Before production build or whenever CSS modules change. |
| `build:js` | `esbuild ./js/script.js --bundle --minify --target=es2018 --format=iife --outfile=./js/script.min.js && node ./scripts/verify-build.mjs js ./js/script.min.js` | Bundles/minifies JS entry to a single browser-ready file and validates output integrity. | Before production build or when JS features change. |
| `build` | `npm run build:css && npm run build:js && npm run build:dist` | Runs full build pipeline (CSS + JS + dist packaging). | For release-ready local build and deployment preparation. |
| `dist:clean` | `node ./scripts/build-dist.mjs --clean` | Removes generated dist output. | Before a clean dist rebuild. |
| `build:dist` | `node ./scripts/build-dist.mjs` | Copies selected files/assets into distribution folder and prepares deployable structure. | After building assets, before deploy artifact handoff. |
| `watch:css` | `postcss ./css/style.css -o ./css/style.min.css -w` | Watches CSS entry and rebuilds minified CSS on changes. | During local CSS development loops. |
| `watch:js` | `esbuild ./js/script.js --bundle --minify --target=es2018 --format=iife --outfile=./js/script.min.js --watch` | Watches JS entry and rebundles minified JS on changes. | During local JS feature development. |
| `img:opt` | `node scripts/optimize-images.mjs` | Generates optimized image outputs from source image directories. | After adding/updating source images. |
| `img:clean` | `node scripts/optimize-images.mjs --clean` | Cleans optimized image outputs generated by the image pipeline. | Before regenerating image outputs from scratch. |
| `img:watch` | `node scripts/optimize-images.mjs --watch` | Watches source image files and regenerates optimized outputs incrementally. | During active content/image editing workflows. |
| `test` | `echo "Error: no test specified" && exit 1` | Placeholder script that intentionally fails. | Not for normal use; replace when adding real automated tests. |
| `check:links` | `node scripts/check-link-integrity.mjs` | Verifies local links/assets across HTML files and `sitemap.xml`. | As a QA gate before publishing or merging. |
| `test:a11y` | `npm exec --yes --package=playwright@1.54.2 --package=axe-core@4.10.3 node scripts/a11y-axe.mjs` | Runs Playwright + axe accessibility checks for configured scenarios. | For accessibility regression checks before release. |
